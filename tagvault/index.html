Request Removal

If you would like to have your server removed from this page, you can:

  • Revoke the invite listed on this page and disable the server widget in server settings. This page removes invalid invites automatically after a few hours.
  • Or send an email with your Server ID. We'll ask you to add a verification text to your server description before removing it.
